偶然间发现了一个很好的方法，详见[Gray code](https://cp-algorithms.com/algebra/gray-code.html)

下面是一个生成以start开始的格雷码的板子，n为二进制位的个数。

```cpp
vector<int> getGray(int n, int start) {
        vector<int> res; 
        for(int i=0;i< 1<<n;i++){
            res.push_back(start^i^i>>1);
        }
        return res;
        
    }
```

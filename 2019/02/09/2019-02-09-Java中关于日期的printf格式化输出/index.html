Java中关于日期的printf格式化输出

发表于 2019-02-09
本文字数： 1.5k 阅读时长 ≈ 1 分钟

printf方法可以很轻易的格式化日期，用两个字母，以%t开头并以下面的其中一个字母结尾。
转换符|说明
—|—|—|—–
a|日期中星期简称
A|日期中星期全称
b|日期中月份的简称
B|日期中月份的全称
C|日期中年份的前两位数
y|日期中年份的后两位数
Y|日期中年份的完整表示
D|“年/月/日”格式
F|“年-月-日”格式
j|一年中的第几天
m|日期中的月份，两位数字表示，不足两位补零
d|日期中的日，两位数字表示，不足两位补零
e|日期中月份的日，不补零
T|“HH:MM:SS” 格式（24小时）
r|“HH:MM:SS PM”格式（12小时）
R|“HH:MM”格式（24小时）
